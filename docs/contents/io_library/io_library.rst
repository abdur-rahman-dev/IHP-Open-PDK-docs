IO and Periphery Library (sg13g2_io)
=====================================

.. _io_library_lbl:

The ``sg13g2_io`` library provides IO pad cells for interfacing the 1.2V core logic domain
with the 3.3V IO domain.

The library files are located in ``$PDK_ROOT/ihp-sg13g2/libs.ref/sg13g2_io/``.
